# Intelligent-Framework-for-QR-Code-Security-and-Authentication
An Intelligent Framework for QR Code Security and Authentication using Curriculum Learning and Ed25519.
Dataset
The dataset used for training the classifier and testing the DWT watermarking consists of 20,000+ augmented QR codes and non-QR code images, along with real life photos of QR codes for better classification.
